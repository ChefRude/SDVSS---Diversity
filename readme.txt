This version of my mod is for compatibility with the "Diverse Stardew" portrait/sprite mod. 

By default, it includes my VanillaSDV portraits for characters not included in that mod: Clint, Evelyn, George, Gus, Jodi, Kent, Lewis, Marnie, Pam, Pierre, Sam, Vincent and Linus

Mod Options via Config.json
A few NPCs have multiple options, which you can configure in the config.json which will load the first time you run SDV with this mod installed. The config.json file shows these options by default. Please look at the images above to see what these options mean.

{
"LuauOnly": "true",
"ElliottVariant": "default",
"GeorgeVariant": "default",
"LinusVariant": "censored",
"DemetriusVariant": "black"
}

Modifying your Config File﻿
Enter the appropriate choice in your config file. Your choices are:

[name of option] - [choice1 (default)]/[choice2] 
LuauOnly - true/false
ElliottVariant - default/beads
GeorgeVariant - default/floaties
LinusVariant - censored/grassskirt
DemetriusVariant - black/albino
